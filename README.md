# MYTRI ENTERPRISES LIMITED Website

A static website for MYTRI ENTERPRISES LIMITED, showcasing sustainable energy products.

## Project Structure
- `index.html`: Main homepage with all sections
- `css/style.css`: Custom styles
- `js/script.js`: Custom JavaScript
- `assets/`: Directory for images (add your product images here)

## Features
- Responsive design using Bootstrap
- Product carousel
- Contact form (static - integrate with Formspree for submissions)
- Smooth scrolling navigation

## How to View
1. Open `index.html` in a web browser.
2. For local development, use a local server (e.g., VS Code Live Server extension).

## Deployment
- Host on Netlify, Vercel, or GitHub Pages for free.
- Replace placeholder images in `assets/` with actual product photos.
- For the contact form to work, integrate with a service like Formspree (https://formspree.io/).

## Customization
- Update content in `index.html`.
- Modify styles in `css/style.css`.
- Add interactivity in `js/script.js`.

## Notes
- Images are placeholders; replace with real ones.
- Form is non-functional on static site; use a backend service for submissions.
